# Water Management System

Description
An intelligent IoT-based system that monitors environmental conditions and optimizes water usage for agriculture.

Tech Stack
- Arduino
- Embedded C
- Sensors (Humidity, Temperature)

Features
- Real-time data monitoring
- Automated irrigation control
- Efficient water utilization
